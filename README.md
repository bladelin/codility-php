# Codility-PHP

Codility-PHP project integration with PHPUnit, PHP CodeCoverage and PHP CodeSniffer.
Using the skeleton to build and created your code.
and get into the habit, good luck.

## Quick Start

### Installation

```bash
$ git clone https://github.com/bladelin

$ cd codility-php

$ make init
`````

### Run Test

```bash
$ make test
```
